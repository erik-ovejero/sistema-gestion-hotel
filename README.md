# sistema-gestion-hotel
Proyecto académico desarrollado para la materia Seminario de Práctica de Informática.

## Descripción

El sistema tiene como objetivo gestionar procesos administrativos de un hotel, incluyendo huéspedes, habitaciones, reservas, pagos y facturación.

## Tecnologías utilizadas

- Java
- Java Swing
- MySQL
- JDBC

## Base de datos

El archivo SQL se encuentra en la carpeta:

database/sistema_hotel.sql
